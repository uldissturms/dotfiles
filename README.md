### setup

```bash
git clone git@github.com:uldissturms/dotfiles ~/.dotfiles
. ~/.dotfiles/init.sh
```

run ```chsh -s /bin/zsh``` to set zsh as default shell

### structure

```
.
├── atom.sh               // install atom plugins
├── vim.sh                // install vim dependencies
├── README.md
├── init.sh
├── macos
│   └── brew.sh
├── oh-my-zsh
│   └── custom
│       └── aliases.zsh
└── windows
    └── settings.json
```
